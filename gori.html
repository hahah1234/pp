펀타스틱코리아
사용언어: spring, java, my-batis, mysql, linux, javascript, jquery, css, html, jsp, bootstrap
펀타스틱코리아는 한국에 방문한 외국의 여행객들에게 여행 상품, 식품, 티켓, 밴 대여 등을 판매자와 연결해주어 상품을 판매하는 사이트입니다. 처음 펀타스틱코리아에 들어왔을때 6년차 CTO 한분과 신입 개발자 두명 있었습니다. 하지만 두명의 개발자 모두 백엔드 개발에 특화되어있어 html css를 작업하는데 어려움이 있어 개발 진행된 프로그램은 부트스트렙 프레임워크 기반으로 페이지들이 개발 되어있었고 부트스트렙 특성상 잘못된 클래스 사용으로 css가 많이 깨져있는 상태였습니다. 입사 초반에 부트스트렙과 css와 html을 공부하며 오류를 수정하여 페이지들의 오류들을 대폭 개선하였습니다. 하지만 회사 자체의 디자이너의 부재와 디자인에 대한 지식이 없어 사이트 디자인개선은 어려움이 많아 아쉬워 하며 디자인의 중요성을 깨달았습니다.
상품페이지의 페이지와 상품의 데이터를 불러오는 쿼리입니다. 관리자시스템인 고리와 DB를 공유하여 관리자시스템에서 정보를 입력하고 입력한 정보에를 불러오는 쿼리를 작성하였습니다.
